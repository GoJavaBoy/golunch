# Vote for restaurant
____

Allow the Authenticated User vote for restaurant.

**URL :** `/profile/restaurants/:pk/`

**URL Parameters :**
`pk=[integer]` where `pk` is the ID of the Restaurant on the server.

**Method :** `PATCH`

**Auth required :** YES

**Permissions required :** None

**Data :** `{}`

### Success Response
____

**Code :** `204 NO CONTENT`
